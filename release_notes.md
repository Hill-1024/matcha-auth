# v2.7.0 交互与视觉体验大更新 ✨

本次更新带来了全方位的 UI 打磨与交互体验升级，致力于为您提供最极致、丝滑的 Material Design 3 (MD3) 体验！

## 🎨 视觉与动效升级 (UI/UX)
- **全局动画统一**：重构了设置页面之间的切换逻辑。一二级页面现在的滑动过渡时间高度统一（0.2s），并且严格区分了前进与后退的方向运动，再也不会有让人“头晕”的碰撞感。
- **动态悬浮搜索栏**：主界面的搜索框重构了物理级弹性动画（Spring Physics）。向下滑动时，搜索框不仅能完美“折叠”进右上角，还加入了智能“迟滞（Hysteresis）”算法彻底告别临界点抽搐；打断动画也能真实继承物理惯性。并且修复了卡片遮挡与抖动的问题。
- **毛玻璃沉浸式顶栏**：主页面现在和设置页一样，拥有了惊艳的 `backdrop-blur-xl` 高斯模糊顶栏，滚动体验更具呼吸感。
- **深色模式专项打磨**：WebDAV 设置等页面的输入框底色、分割线、描边等细节现已完美遵循 MD3 深色模式规范。

## 📱 完美的响应式适配
- **重制 ActionSheet 菜单面板**：
  - **移动端**：移除了最大宽度限制，现在它是 100% 贴合屏幕的沉浸式 Bottom Sheet，并完美兼容全面屏底部安全区（Safe Area），告别误触。
  - **宽屏/PC端**：利用实时媒体查询重构了表现形态。在宽屏上它会自动升格为带描边的 MD3 居中对话框（Dialog），且自动适配进出场的缩放与渐变动画，彻底修复了原先在屏幕上“滞留”干等的尴尬 Bug。

## ⚙️ 底层优化
- **包管理器换新**：项目构建已全面迁移至 `pnpm`，不仅锁定依赖更加严格，更是带来了极致的安装与构建速度。
