# v2.7.1 交互体验热修复 🐛

本次更新主要针对 v2.7.0 中搜索栏的动画交互逻辑进行了“降维打击”式的修复，确保在所有极端场景下都能获得符合直觉的丝滑体验。

## 🐛 修复与优化
- **根治搜索框折叠动画抽搐**：重构了列表底层布局逻辑，通过引入物理级别的“冗余滚动空间 (Overscroll Padding)”，彻底杜绝了在令牌数量较少（短页面）时，由于搜索框收起导致的高度坍塌和“无限死循环抽搐” Bug。
- **修复搜索框被顶栏遮挡问题**：去除了之前为了防止抽搐而妥协的“短页面禁止收起”限制。现在即便在只有少数令牌的情况下下拉页面，搜索框也能正常地、优雅地收起，而不会诡异地滑动到半透明顶栏下方发生重叠。
- **更智能的防误触与键盘管理**：优化了滑动时收起系统键盘的触发逻辑。只要处于搜索输入状态，页面发生哪怕最轻微的下拉，都能毫无延迟地瞬间触发键盘收缩，带来极致原生流畅感。
